# PWA Icons

This directory contains placeholder icons for the Holiday Planner PWA.

## Required Icons

Replace these SVG placeholders with proper PNG files before production:

| File | Size | Purpose |
|------|------|---------|
| icon-72x72.png | 72x72 | Android icon |
| icon-96x96.png | 96x96 | Android icon |
| icon-128x128.png | 128x128 | Android icon |
| icon-144x144.png | 144x144 | Android icon |
| icon-152x152.png | 152x152 | iOS icon |
| icon-192x192.png | 192x192 | Android icon |
| icon-384x384.png | 384x384 | Android splash |
| icon-512x512.png | 512x512 | Android splash (maskable) |
| apple-touch-icon.png | 180x180 | iOS home screen |
| badge-72x72.png | 72x72 | Notification badge |

## Generating Icons

Use a tool like [pwa-asset-generator](https://github.com/nickvidal/pwa-asset-generator) or design the icons in Figma/Sketch and export at all sizes.

Theme color: #8B5CF6 (Purple)
